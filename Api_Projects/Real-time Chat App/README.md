# 💬 ChatStream - Real-time Chat Application

A full-stack real-time chat application built with Node.js, Express, Socket.io, and vanilla JavaScript.

## 🚀 Features

- **Real-time Messaging**: Instant message delivery using WebSockets
- **Multiple Rooms**: Create or join different chat rooms
- **Private Messaging**: Direct message other users with `/pm @username message`
- **Typing Indicators**: See when others are typing
- **User List**: Live online users in each room
- **Message History**: Last 50 messages persist in each room
- **Emoji Picker**: Quick emoji insertion
- **Responsive Design**: Works on desktop and mobile
- **Profanity Filter**: Automatic content moderation
- **Theme Toggle**: Dark/Light mode support

## 🛠️ Tech Stack

- **Backend**: Node.js, Express, Socket.io
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Real-time**: WebSocket (Socket.io)
- **Utilities**: Moment.js (time formatting), bad-words (content filter)
