#  DevStudio - Online Code Editor

DevStudio is a modern **online code editor** where you can write and preview **HTML, CSS, and JavaScript** in real time.

It is designed with a clean UI and smooth experience, similar to tools like CodePen.

## ✨ Features

- 🧑‍💻 Live HTML, CSS, JS Editor
- ⚡ Instant Preview (Run Button)
- 🎨 Beautiful UI with Tailwind CSS
- 🌙 Dark Theme (Dracula)
- 🔁 Reset Code Option
- 📱 Responsive Layout


## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **Tailwind CSS**
- **CodeMirror Editor**
